# 09 — Execution Form

Use this as a checklist. Do not jump ahead.

## Phase 1 — Campaign Foundation

- [ ] Confirm campaign name.

## Phase 2 — Story Development

- [ ] Finalize the soccer father/son hero story.
- [ ] Write 3–5 minute hero video voiceover.
- [ ] Break hero video into 8–12 scenes.
- [ ] Create character bible for father.
- [ ] Create character bible for son.
- [ ] Create setting bible: home, soccer field, stadium, tree.
- [ ] Create shot list for each scene.
- [ ] Create 7 short-form cutdowns from the hero story.
- [ ] Create 4 alternate family stories.

## Phase 3 — AI Visual Production

- [ ] Generate still reference images for father, son, home, field, stadium, tree.
- [ ] Lock visual style: cinematic, warm, realistic, emotional, premium.
- [ ] Generate each scene as short 5–8 second clips.
- [ ] Keep character consistency using reference frames.
- [ ] Generate B-roll shots: hands, cleats, tree, photos, work boots, stadium lights.
- [ ] Export all clips in 16:9 and/or 9:16.
- [ ] Organize clips by scene number.
- [ ] Review for weird hands/faces/logos/text.
- [ ] Replace any broken AI footage.
