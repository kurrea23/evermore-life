# SHORT FORM VIDEO HOOKS
## Evermore Life — TikTok / Instagram Reels / YouTube Shorts
**Format:** 10 hooks + 30-second script outlines
**Platform:** TikTok primary, repurposed to Reels + Shorts
**Version:** 1.0 — Production Draft

---

## HOW TO USE THIS DOCUMENT

Each entry contains:
1. **The Hook** — The exact first 3 seconds of on-screen text or spoken line. This is the scroll-stopper.
2. **ICP Target** — Which ideal client this speaks to
3. **30-Second Script Outline** — Shot-by-shot or line-by-line breakdown for the full clip
4. **CTA** — How the video ends (drives to hero video, funnel page, or "link in bio")
5. **Production notes** — How to make it

**General production approach for all short-form:**
- Vertical format, 9:16
- First 3 seconds determine everything — nail the hook
- Captions on every video (85% of TikTok watched without sound)
- No logos in first 3 seconds — earn attention before branding
- End with a clear, single CTA — never two

## PLATFORM POSTING STRATEGY

**TikTok:** Post 3–5x per week. Mix emotional (hooks 1, 4, 6, 8) with educational (3, 5, 7, 9). Hook 10 only in retargeting.

**Instagram Reels:** Post 3x per week. Same content, slight caption adjustments for Instagram tone.

**YouTube Shorts:** Post 2x per week. Prioritize educational hooks (3, 7, 9) — YouTube Shorts audience is more info-hungry.

**A/B Testing Protocol:**
- Run each hook pair simultaneously in the first 2 weeks: Hook 1 vs 6 (ICP1 emotional), Hook 4 vs 8 (ICP2 emotional), Hook 2 vs 10 (urgency vs inspiration)
- After 7 days: pause lower performer, double budget on winner
- Educational content (3, 5, 7, 9): run all simultaneously, rank by saves + shares
